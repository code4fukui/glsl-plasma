# glsl-plasma

Three.jsとGLSLによるプラズマ効果のデモです。

## デモ
[デモページ](https://code4fukui.github.io/glsl-plasma/)

## 機能
- 時間とともにプラズマ効果が変化します
- 表示パラメータをスライダーで調整できます
- ランダムボタンで自動的にパラメータがランダム化されます

## 必要環境
- Web ブラウザ

## 使い方
ブラウザでデモページを開いてください。スライダーを動かしてプラズマ効果を変更できます。ランダムボタンを押すと自動的にパラメータがランダム化されます。
